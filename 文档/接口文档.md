## 1.用户

### 创建新用户

#### Reuqest

- Method: **POST**

- URL: ```/user```

- Headers:    Content-Type:multipart/form-data

- Body:

  ```
  phone      111111
  username   newguy
  password   112322
  image      file
  ```

#### Response
- Body
```
{
  "code": 200,
  "message": "new user has created"
}
```

### 更新用户信息

#### Reuqest

- Method: **PUT**

- URL: ```/user```

- Headers:    Content-Type:multipart/form-data

- Body:

  ```

  username   newguy
  password   112322
  image      file
  ```

- Body
